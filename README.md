## *:rocket: Instalation*
#### STEP 1 Create DB
Create Database `db_laravel9`
#### STEP 2 Config ENV
Change file `env.development` to `.env`
#### STEP 3 Set Up
Open Terminal in your folder project and run command:
```
composer install
```
```
php artisan migrate
```
```
php artisan db:seed
```

#### STEP 4 Running The Project
```
php artisan serve
```
and put `http://127.0.0.1:8000/` in your browser.
